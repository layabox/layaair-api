Static create

  • create(source: Texture2D | Texture, x: number, y: number, width: number, height: number, offsetX?: number, offsetY?: number, sourceWidth?: number, sourceHeight?: number): Texture
    • Defined in laya/resource/Texture.ts:92

    根据指定资源和坐标、宽高、偏移量等创建 Texture 对象。

    Parameters

    • source: Texture2D | Texture

      绘图资源 Texture2D 或者 Texture对象。

    • x: number

      起始绝对坐标 x 。

    • y: number

      起始绝对坐标 y 。

    • width: number

      宽绝对值。

    • height: number

      高绝对值。

    • Default value offsetX: number = 0

      X 轴偏移量（可选）。 就是[x,y]相对于原始小图片的位置。一般都是正的，表示裁掉了空白边的大小，如果是负的一般表示加了保护边

    • Default value offsetY: number = 0

      Y 轴偏移量（可选）。

    • Default value sourceWidth: number = 0

      原始宽度，包括被裁剪的透明区域（可选）。

    • Default value sourceHeight: number = 0

      原始高度，包括被裁剪的透明区域（可选）。

    Returns Texture

    Texture 对象。
