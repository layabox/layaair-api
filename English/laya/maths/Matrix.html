Classpublic class Matrix
InheritanceMatrix Object

Matrix Class represents a transformation matrix, which determines how to map points from one coordinate space to another.

You can perform different graphical transformations on a display object by setting it. Matrix The attribute of the object. Matrix The object is applied to the Transform object Matrix Property, and then applies the Transform object as the transform property of the display object. These conversion functions include translation (x and Y relocation), rotation, scaling and tilting.
